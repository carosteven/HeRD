# Setup
## Install Submodules
```bash
git submodule add https://github.com/IvanIZ/BenchNPIN submodules/BenchNPIN
cd submodules/BenchNPIN
pip install -e .
cd ../..
```

```bash
git submodule add https://github.com/IvanIZ/spfa.git submodules/spfa
cd submodules/spfa
pip install -e .
cd ../..
```
